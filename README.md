# Cornerstone Design System

## Prerequisites

### Figma plugin for Claude

The `/pull-tokens` command syncs design tokens from Figma into this repo and requires the Figma plugin for Claude Code.

To install:

1. Run `/plugins` in Claude Code to open the plugin marketplace
2. Search for **Figma** and install it
3. Authenticate with your Figma account when prompted
4. Restart Claude Code — the plugin is discovered at session init

Once installed, run `/pull-tokens` in Claude Code from this repo.
